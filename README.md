# 家庭财务管理系统

---

## 项目简介

### 软件定位
家庭财务管理系统是一款面向家庭的轻量级财务管理工具，帮助家庭成员便捷地记录、统计和管理日常收支，实现财务数据可视化，为家庭财务决策提供数据支持。

### 基本功能
- **收支记录**：快速记录收入和支出，支持分类标签、备注说明
- **数据统计**：按时间、类别、成员等多维度统计分析财务数据
- **预算管理**：设置月度预算，实时监控预算执行情况
- **资产查看**：查看家庭资产状况和资金流向
- **多用户协同**：支持家庭成员共享数据，信息实时同步
